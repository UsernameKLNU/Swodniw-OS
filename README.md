# 特性
```
Swodniw OS 1.0 是 Swodniw 的第1个主要版本，也是Windows Quantum阵容的第12个主要版本。
可支持800x600到16K显示器，缩放由 Windows 11 的 100% 到 225% 升级到了 1% 到 5000%。
可运行 16 位应用到 64 位应用。
在此版本中，所有的系统文件夹及文件被合并到了文件夹“Swodniw”，
“Swodniw”更名为“SystemUniversal”，以确保 Swodniw 系统支持所有电脑。
跨平台支持可支持Linux，Mac，Windows和Swodniw OS。
此外，我们不再引用术语“PC”，因为Swodniw现在是具有Quantum 11内核的完全通用系统。
此设备也已重命名为“此量子设备”，以完成向新量子内核的过渡。
```
